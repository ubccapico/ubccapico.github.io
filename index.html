The UBC Canvas API Community (CAPICO) is an open community for anyone at UBC, including academic and professional staff, students and other stakeholders, to share practices and learn about the Canvas API and how it may be used. We maintain an open repository of our scripts spanning multiple languages at https://ubccapico.github.io/

The Canvas virtual learning environment provides RESTful API (Application Programming Interface) for interacting with most aspects of the system through authenticated calls to pull and push data to and from the system. Leveraging this for pedagogical gains, learning analytics and course management is possible and is currently being explored through the groups' work. If you'd like to learn more about the Canvas API, please visit https://canvas.instructure.com/doc/api/index.html

NOTE: There are currently no dedicated staff at UBC for developing LTI Apps (Learning Technology Integration) or command-line scripts, so CAPICO exists to provide others with guidance and support in their own implementations and explorations.

We typically meet on the third thursday of every month, so if you're interested in attending, or have any questions about the API, security or privacy in its use, or if you're an instructor and have ideas on how you may need support in its use, please contact the current co-chairs of the community Barish Golland, or Stoo Sepp.
